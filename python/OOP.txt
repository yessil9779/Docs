### Декораторы
@classmethod -- Используйте для реализации методов, которые работают со всем классом(а не с конкретными экземплярами класса) 
def validate(cls,arg):...

@staticmethod -- В отличие от обычного метода или класса, статический не получает дополнительный аргумент self или cls. Это обычная функция которая определяется внутри класса
def norm(x,y):...

--Обычный метод в классе
def deposit(self, amount):
    self.balance += amount
--------------------------------------------------------------------------
### Мехонизм инкапсуляции
* attribute (без одного или двух подчеркиваний вначале) - публичное свойство (public)
* _attribute (с одним подчеркиванием) - режим доступа protected (служит для общения внутри класса и во всех его дочерних классах) 
* __attribute (с двумя подчеркиваниями) - режим доступа private (служит для обращения только внутри класса)
